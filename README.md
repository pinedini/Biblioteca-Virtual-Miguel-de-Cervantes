# 📚 Biblioteca Virtual Miguel de Cervantes

## Descripción

La Biblioteca Virtual Miguel de Cervantes es un sistema de base de datos diseñado para gestionar la información de una biblioteca digital. Su objetivo es facilitar el almacenamiento, organización y consulta de datos relacionados con libros, autores, usuarios y préstamos, permitiendo una administración eficiente de la información.

Este proyecto fue desarrollado con fines académicos para poner en práctica conceptos de diseño e implementación de bases de datos utilizando SQL.

---

## Objetivos

- Gestionar el catálogo de libros.
- Registrar información de autores y editoriales.
- Administrar los usuarios de la biblioteca.
- Controlar el préstamo y devolución de libros.
- Facilitar consultas mediante sentencias SQL.

---

## Tecnologías utilizadas

- MySQL
- SQL
- XAMPP
- GitHub

---

## Estructura de la base de datos

La base de datos está compuesta por tablas relacionadas que almacenan información como:

- Libros
- Autores
- Editoriales
- Usuarios
- Préstamos
- Categorías

Cada tabla cuenta con claves primarias y foráneas que garantizan la integridad de los datos.

---

## Funcionalidades

- Crear y administrar registros de libros.
- Registrar nuevos usuarios.
- Gestionar préstamos y devoluciones.
- Consultar información mediante sentencias SQL.
- Actualizar y eliminar registros cuando sea necesario.

## Instalación

1. Clonar este repositorio.

```bash

```

2. Abrir MySQL desde XAMPP.

3. Crear una nueva base de datos.

4. Importar el archivo `.sql` incluido en el proyecto.

5. Ejecutar las consultas según sea necesario.
